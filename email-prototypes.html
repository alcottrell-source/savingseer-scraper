Three ways to make sale day feel like an occasion

The current emails are cream-on-cream text with system fonts. All three of these load the real brand type (Playfair Display + Inter, Georgia/Arial fallbacks), use the true site palette, and put a picture in a frame — the shopping-centre scene is hand-drawn in the brand colours, and a real photo of each centre drops into the same frame the moment we source one. Same Peak-day data in all three; what differs is the mood: gallery, day-trip, or night-out.

CThe Night Tide — signal-green drama

The app's dark-surface language, finally in an email: bark canvas, the neon tide green the site uses for the live line, a huge glowing score, and the centre lit up at night inside a neon frame. Loudest and most urgent — peak day as an event you don't miss.

Prototype only — no production change; notify-high-tide is untouched until a direction wins. Palette + type are verbatim from index.html tokens (cream / bark / leaf / amber / tide-neon, Playfair Display + Inter with Georgia/Arial fallbacks — Google-font loading works in Apple Mail and falls back cleanly elsewhere). The scenes are hand-drawn SVG in the brand palette; for the real send each gets exported to PNG and hosted from the repo (Vercel serves the root), and a genuine centre photograph can drop into the same frame per centre. Porting the winner means table-safe markup + inlined styles in renderHighTideEmail — a follow-up ticket.
